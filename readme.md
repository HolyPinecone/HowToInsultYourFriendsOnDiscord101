## Getting started

1. Install npm packages
    ```bash
    $ npm install
    ```
2. Copy the `.env.example`
    ```bash
    $ cp .env.example .env
    ```
3. Change the variables in the `.env`
4. Start the development server
    ```bash
    $ npm run dev
    ```
